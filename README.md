# lindsey
A simple browser for the Link2, written in Qt.

## Requirements
* Qt 5.4 or higher
* CMake 2.8.11 or higher

## Building (OS X and Linux)
	cd lindsey
 	mkdir build
 	cd build
 	cmake ..
 	make
